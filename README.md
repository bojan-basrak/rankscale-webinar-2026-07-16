# Rankscale Webinar 2026

Materials for the Rankscale API webinar — a visually consistent series of HTML docs
covering how to use the Rankscale AI brand-visibility API.

## What's here

_(This section will grow as materials are added.)_

- Home page linking the report artifacts and API usage examples
- Report: **MCP vs API**
- Report: **AI Engine Consistency**
- API usage-example docs (visibility, Brand Rank, citations, sentiment, credits, bulk export)

## Notes

- All pages are self-contained (inlined CSS/JS) so they work as standalone files.
- Shared design system: cool paper/ink palette, teal `#0f766e` / amber `#b45309` accents,
  serif display type, theme-aware (light + dark).

## ⚠️ No secrets in this repo

This is a **public** repository. API keys and credentials must never be committed here —
they live in local environment variables only. See `.gitignore`.
